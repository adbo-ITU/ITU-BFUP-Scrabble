## Usage
Firstly, add the ID of your Sheets document in the file `.env`:
```
SHEET_ID=<insert sheet id here>
```
Make sure that link sharing has been enabled for the sheet.

## Installation
Haven't tested, but I think it's something like
```
python -m pip install python-dotenv pandas
```
